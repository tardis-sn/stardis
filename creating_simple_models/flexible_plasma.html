These temperatures come from the model we read in, which represent the sun. However, for demonstration purposes, let‚Äôs overwrite these temperatures to go from 3,000 to 10,000 K. This can be done by simply overwriting the object‚Äôs temperatures attribute. For ease of use, let‚Äôs match the same number of depth points as the existing one, so we don‚Äôt have to worry about shape mismatches for other parts of the model. We‚Äôll also overwrite the density so we can examine how molecular formation changes purely as a function of temperature.
